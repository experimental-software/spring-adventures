# Contributing

## Commit messages

Commit messages should follow the [Conventional Commits](https://www.conventionalcommits.org/) guidelines
and include a reference to a [GitHub issue](https://github.com/experimental-software/spring-adventures/issues) in the message body.

```
<type>(<scope>): <subject>

GH-42
```

- type: see https://github.com/angular/angular/blob/22b96b9/CONTRIBUTING.md#type
- scope: name of the affected application module (optional)
- subject: short summary of the changes

**Also see**

- https://www.conventionalcommits.org/en/v1.0.0/#summary
- https://www.baeldung.com/ops/git-commit-messages
